

## 💻 Conforme proposto no bootcamp sobre Copilot e IA da DIO, este projeto consiste em, através de dois prompts (que podem se tornar agentes, mas no caso foram apenas utilizados como prompts), usar a inteligência artificial para descobrir qual carreia em tecnologia seguir.

O primeiro prompt faz a IA agir como entrevistador para saber qual carreira sugerir. <br>
O segundo serve para planejar a carreira, com sugestões de conteúdo para estudo.
