

## 💻 Conforme proposto no bootcamp sobre Copilot e IA da DIO, este projeto consiste em, através de dois prompts, usar a inteligência artificial para descobrir qual carreia em tecnologia seguir.

O primeiro prompt faz a IA agir como entrevistador para saber qual carreira sugerir. 
O segundo serve para planejar a carreira, com sugestões de conteúdo para estudo.
