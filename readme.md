# 🚀 Planejamento de Carreira: Desenvolvedor Front-end (Resultado prompt 2)

Este documento detalha o roadmap de estudos personalizado e o diagnóstico de carreira para a minha transição para a área de tecnologia.

---

## 📅 Fase 1: Roadmap de 150 Dias (Planejamento)

**Meta:** Transição de carreira interna | **Dedicação:** 14 horas/semana

### Mês 1: Fundamentos & O Mundo Git
- **Semanas 1-4:** HTML/CSS Avançado e comandos essenciais de Git (`commit`, `push`, `branch`). Foco em subir os primeiros projetos para o GitHub.

### Mês 2: Lógica & Interação
- **Semanas 5-8:** JavaScript (DOM, Funções, Objetos). Início do hábito de commits diários para construção do portfólio.

### Mês 3: APIs & Dados Dinâmicos
- **Semanas 9-12:** Consumo de APIs com Fetch/Async-Await. Integração de funções de IA simples em interfaces web.

### Mês 4: Frameworks (React)
- **Semanas 13-16:** Introdução ao React.js e arquitetura de componentes. Uso de Git Flow para organizar o desenvolvimento.

### Mês 5: Projeto Final & Refinamento
- **Semanas 17-20:** Desenvolvimento e deploy do projeto **FinancIA**. Documentação técnica dos repositórios e preparação para entrevistas internas.

---

## 🚀 Projeto de Portfólio: "FinancIA"
* **Escopo:** Dashboard financeira responsiva com insights gerados por IA.
* **Destaque:** Interface de alta fidelidade focada em UX, aproveitando o background criativo em Publicidade e Propaganda.
